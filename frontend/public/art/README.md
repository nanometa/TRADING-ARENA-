# Dossier des illustrations (art)

Dépose ici tes illustrations Ritual, nommées par numéro :

```
public/art/1.png
public/art/2.png
public/art/3.png
...
public/art/40.png
```

Elles apparaissent automatiquement aux emplacements prévus dans le site
(hero, sections, galerie). Tant qu'un fichier est absent, un placeholder
numéroté élégant s'affiche à sa place — aucun code à modifier.

Formats supportés : `.png` (par défaut) ou `.jpg` (passer `ext="jpg"`).
Conseil : images au format vertical (4:5) ou carré pour un rendu optimal.
